> [Vega][] and [Vega-Lite][] boilerplate generator for [Yeoman][].

## Installation

First, install [Yeoman][] and generator-veeg using [npm][] (we assume you have pre-installed [node.js][]).

```sh
npm install -g yo
npm install -g generator-veeg
```

Then generate your new project:

```sh
yo veeg
```

## Help

```sh
yo veeg --help
```
